# Train Yolo8
Landing template for detect any object width python

## Required Instalation
1. `pip install labelImg`
2. `pip install opencv-python`
3. `pip install ultralytics`


## New scaning object
cmd
```cmd
$ labelImg
```


## Training object
cmd
```cmd
$ python train.py
```

## Running test
video:
```cmd
$ python test_video.py
```

image:
```cmd
$ python test.py
```
